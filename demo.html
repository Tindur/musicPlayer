jQuery Music Player

This is the results from an assignment in WEPO 2013 at Reykjavik University.

Note that including two or more musicplayers will result in strange behaviour.

Features

HTML 5

This Music Player uses HTML 5 audio element to play local mp3 files or files from some URL.

Album artwork from Echonest

Music Player comes with built in Album Artwork fetcher that compares the filename with the huge database from Echonest. The filenames should be very well structed. For example: Britney Spears - Toxic.mp3

Local files and URL

Music Player doesn't care whether you are playing music from your hard drive or loading the song from the Internet. Just throw something in the playlist array and start diddleing to some sweet tunes!

Sponge Bob!

SpongeBobbyFy the player to get some real groove going on. We highly reccomend cannabis + tequila intake hours before trying this.

Dependencies

This plugin requires only jQuery to work and was tested with version 1.9.1

Installation

Reference jQuery, musicPlayer.js and musicPlayer.css in your the HTML header

Using the Music Player

Basic player

Create a div element and set the id or class to whatever you want

In your script file, make an array with mp3 filenames, and send it to the musicPlayer 

			var playlist = ["Britney Spears Toxic.mp3",
						"Britney Spears Radar.mp3"];
			$("#musicPlayerDemo").musicPlayer(playlist);

That will result in

With playlist

				var playlist = ["Britney Spears Toxic.mp3",
							"Britney Spears Radar.mp3"];
				$("#musicPlayerDemo").musicPlayer(playlist, {playlist: true});

Songs from URLs

				var playlist = ["http://previews.7digital.com/clips/34/9522.clip.mp3",
							"Britney Spears Radar.mp3"];
				$("#musicPlayerDemo").musicPlayer(playlist, {playlist: true});

SpongeBob

To spongebobbify , add this to your script 

				var playlist = ["http://previews.7digital.com/clips/34/9522.clip.mp3",
							"Britney Spears Radar.mp3"];
				$("#musicPlayerDemo").musicPlayer(playlist, {spongebob: true});

And then in your header, change the CSS location to the Spongebob CSS location

Options

  • playlist - Default False. If set as True, it will append a little playlist just beneath the player.

  • autoplay - Default False If set as True, it will annoy the user. Please don't use this.

  • defaultVolume - Default 100 which is also the maximum volume you can squeese out of the player.
